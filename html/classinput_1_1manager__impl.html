Detailed Description

The base class for input manager implementation.

Note
In case you want to share the same implementation for several input::managers, you have to call set_manually_updated(true), and update the manager yourself. Else, it will be updated by each input::manager (which may not be desirable).

◆ toggle_mouse_grab()

virtual void input::manager_impl::toggle_mouse_grab ( )
pure virtual

Toggles mouse grab.

When the mouse is grabbed, it is confined to the borders of the main window. The actual cursor behavior when reaching those borders is of no importance : what matters is that relative mouse movement is always aquired, i.e. the mouse is never blocked. The mouse is not grabbed by default.

Implemented in input::ois_manager, and input::sfml_manager.
