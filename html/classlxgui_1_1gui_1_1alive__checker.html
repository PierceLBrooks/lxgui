Detailed Description

Utility class for safe checking of widget validity.

To use this class, construct an instance of alive_checker with any object you wish to monitor. Then use the object. Then use alive_checker::is_alive() to check if the object is still alive.

Note
This class will not be able to notice that an object is not alive if the object was already destroyed before the alive_checker instance is created.
